Statement of Environment
This website is hosted on Github Pages, which has servers based in the United States. Some mentioned locations are "Seattle", and "one on each cost of the states". I pay full respect to the traditional and current owners of the land. Read More

With some eco-awareness in mind, I aim to avoid 3rd party javascript runners, sites and fonts, which should get less servers involved, and in turn reduce the energy impact that is required to run this site.
This site was built on pure HTML, CSS and JS, and does not need to request resources from other servers to run, excluding embedded content.

All credits go to the original artists, I do not claim to have created any of these wonderful places. Do go ahead and look into them if any fancy you!
